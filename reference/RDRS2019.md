# Arquivo da AIH - RS, 2019

Internações hospitalares pagas pelo SUS no RS em 2019.

## Usage

``` r
RDRS2019
```

## Format

Banco de dados com amostra aleatória de 10.000 internações dos "arquivos
reduzidos" (RD\*.DBC) das Autorizações de Internação Hospitalar (AIH)
pagas pelo SUS em hospitais do RS em 2019 ("ano de competência" = 2019).
O banco é formado pela junção (adição de linhas) de cada um dos arquivos
RDRS1901.DBC, ..., RDRS1912.DBC.

## Source

Bases de Dados do Sistema de Informações Hospitalares do SUS
(BD-SIH/SUS)
