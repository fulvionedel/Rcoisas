# População por sexo e faixa etária. Municípios gaúchos, 2019.

População por sexo e faixa etária. Municípios gaúchos, 2019.

## Usage

``` r
POPRS2019
```

## Format

Banco de dados com estimativas populacionais por sexo e faixa etária
para os municípios do Rio Grande do Sul em 2019. Criado com a função
[`popbr2000_2021`](https://fulvionedel.github.io/csapAIH/reference/popbr2000_2021.html),
que por sua vez usa os dados tabulados do DATASUS e importados pelo
pacote `brpop`, de Raphael Saldanha.
